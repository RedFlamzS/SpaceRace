# SpaceRace

## About

Space Race site made for history

## Technology Stack

This project is built with web technologies:

- **HTML** (87.5%) - Structure and content markup
- **CSS** (11.1%) - Styling and responsive design
- **JavaScript** (1.4%) - Interactive features and functionality

## Features

- Historical timeline of space race events
- Information about key missions and achievements
- Educational content for learning about space exploration
- Responsive design for accessibility across devices
